## system

-   Serial Port communication

    -   DIP(00010)
    -   Baud rate: 115200
    -   Data bits: 8
    -   Parity: None
    -   Stop bits: 1
    -   Flow control: None

-   Boot from SD card

    -   DIP(00110)
